## How To
### Install
```
$ git clone https://github.com/vbextreme/gnome-overview-conf.git
$ cd gnome-overview-conf
$ sudo ./install
```

### Uninstall
```
$ sudo ./uninstall
```

### Usage
for view all command</br>
```
$ gnome-overview-conf help
```
after setting value reload shell with</br>
```
ALT+F2 
r
```
